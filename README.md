## Projects

**[Arc](https://github.com/VxidDev/Arc)** - A lightweight programming language written in C, featuring a custom lexer, parser, and bytecode VM. Faster than CPython on 6/8 benchmarks.

**[Nebula](https://github.com/VxidDev/Nebula)** - A minimal async web framework for Python built on native ASGI. Hits 48k req/s under load.

**[ArcOS](https://github.com/VxidDev/ArcOS)** - A 16-bit real-mode operating system written entirely in x86 Assembly, with a multi-stage bootloader and custom syscall interface.

**[EduDuck](https://github.com/VxidDev/EduDuck)** - An AI-assisted study app that turns notes and PDFs into quizzes, summaries, and flashcards via OCR and LLM pipelines.

**[Axionetd](https://github.com/VxidDev/Axionetd)** - A lightweight HTTP server framework written in C from scratch using raw sockets.
